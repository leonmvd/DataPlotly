Contourdiagram¶

Hier kun je de handleiding voor iedere parameter van het countourdiagram vinden. Indien je meer generieke informatie zoekt, zie DataPlotly Standaard gebruik

Diagram eigenschappen¶

Layer: de keuzelijst toont alle in QGIS ingeladen vector lagen

X Field: het X veld

Y Field: het Y veld

Contour Type: kies het countourtype:

  1. Fill: het diagram wordt overal gevuld:
  1. Heatmap: een hittekaart achtige grafiek wordt gemaakt:
  1. Only Lines: alleen de contourlijnen zullen worden getoond:

Show Lines: vink deze optie uit als er geen contourlijnen moeten worden toegevoegd aan het diagram

Color Scale: kies tussen verschillende kleurenschema’s

Diagram aanpassingen¶

Show Legend: toon de legenda van het huidige diagram

Plot Title: titel van het diagram

Legend Title: de titel van de legenda
