DataPlotly come provider di Processing¶

Dalla versione 1.4, DataPlotly può essere usato come provider di Processing grazie al lavoro di Michaël Douchin of 3Liz.

Questa funzione apre le porte ad infinite possibilità:

  • usare tutte le funzioni di Processing con DataPlotly
  • creare grafici in serie con un semplice click
  • aggiungere DataPlotly al modellatore grafico di Processing

Attivare DataPlotly per Processing¶

La prima cosa da fare è attivare Processing e aggiunger DataPlotly come provider. Se DataPlotly non è visibile in Processing bisogna attivarlo nella scheda Processing del menu ``Impostazioni -> Opzioni`.

uso semplice di DataPlotly¶

L’algoritmo Crea un grafico generico è una versione semplificata di DataPlotly: questo significa che non saranno disponibili tutte le personalizzazioni, ma si possono comunque creare grafici fantastici.

L’interfaccia è molto semplice e la creazione di un grafico è altrettanto facile:

  • scegli il vettore
  • scegli il tipo di grafico
  • scegli le opzioni aggiuntive (titolo, asse X, asse Y e colore)

Si ottengono 2 risultati:

  • file html del grafico
  • file json con tutte le specifiche del grafico

Il grafico è caricato in Processing -> Visualizzatore risultati:

Con un semplice doppio click sulla voce del menu il grafico si aprirà nel browser predefinito:

Se vuoi salvare anche il file json devi specificare un percorso preciso nella finestra dell’algoritmo.
